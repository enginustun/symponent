Symponent Documentation v1.0

This is a JavaScript UI library that allows you to create reactive UIs with a very small learning curve.
Symponent does not use virtual DOM nor two-way data binding.
It aims to render only related elements if it is necessary.

Installation

Firstly, you can download latest relase from here, and you can see older versions from here.
And then include script tag you downloaded to your html page. 

<script src="{yourFolderStructure}/symponent.js"></script>

Compatibilities

Symponent supports IE9 and above, because it uses ECMAScript 5 features.
