symponent Documentation Version 1.0

Introduction

symponent is a light-weight but very powerful JavaScript UI library that allows you to create reactive UIs with a very small learning curve.
It aims to render only related elements if it is necessary.

Download

Download symponent production release to finish your works as soon as possible or source code to contribute this awesome library.

Compatibilities

symponent supports IE9 and above, because it uses ECMAScript 5 features.

Prerequisites

This guide assumes intermediate level knowledge of HTML, CSS, and JavaScript. If you think you are not familiar with them, it might not be a good idea to start learning this library. After experiencing the basics you might be symponent master. Prior experience with other frameworks helps, but is not required.

Usage

Getting Started

Include script tag you downloaded to your html page.

<script src="{yourFolderStructure}/symponent.js"></script>

Create First Component

symponent supports only JavaScript syntax yet. The smallest symponent example looks like below:

  • JavaScript
  • HTML
  • CSS
sym.createComponent(
    sym.createElement('div', {}, 'Hello symponent!'),
    'container-element-id'
);
<div class="container-element-id"></div>

Result

Models and Template Syntax

symponent allows you to use models in elements with curly brackets {modelName.propertyName}. As you can understand that you should pass model parameter to the element created for using it in element.

  • JavaScript
  • HTML
  • CSS
var userObject = { name: 'John', surname: 'Doe' };
sym.createComponent(
    sym.createElement('div', { model: { user: userObject } }, 'Hello {user.name} {user.surname}!'),
    'container-element-id'
);
<div class="container-element-id"></div>

Result

Interactivity

Interactive user inputs can be defined easily.

  • JavaScript
  • HTML
  • CSS
var userObject = { name: 'John', surname: 'Doe' };

sym.createComponent(
    sym.createElement('div', { model: { user: userObject } }, 
        'Hello {user.name} {user.surname}!',
        sym.createElement('input', { value: '{user.name}' })
    ),
    'container-element-id'
);
<div class="container-element-id"></div>

Result
